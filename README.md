# Text-to-Morse
This is a python text-based program that converts any string into morse code and morse code into string

To run this program (if you are a non-programmer) click on the "82_Day_Text_To_Morse.py" file -> click on raw download (save on desktop, optional), then on desktop open command prompt (Win + R) type "cd (Your file location) -> hit enter -> type "python 82_Day_Text_To_Morse.py".If you get an error like python is not recognized, you may need to install Python first and restart your Command Prompt. Program is running.
