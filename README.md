# Text-to-Morse Converter

A simple Python text-based program that converts any string into Morse code and Morse code back into text.

## Features
- Convert text to Morse code
- Convert Morse code back to text
- Easy to run and use in the terminal/command prompt

## Requirements
- Python 3.10 or higher

## How to Run

### Step 1: Get the Program File
1. Open this repository and find the file named `82_Day_Text_To_Morse.py`.
2. Click on the file.
3. On the top-right of the code area, click **Raw**.
4. Right-click the page → select **Save As…** → save it to your Desktop (or any folder you prefer).

### Step 2: Open Command Prompt
Press `Win + R` on your keyboard.  
Type `cmd` and hit Enter — this opens the Command Prompt window.

### Step 3: Go to the File’s Location
In the Command Prompt, type:
```bash
cd Desktop
```
Then type:
```bash
python 82_Day_Text_To_Morse.py
```
